# Skill: Bxc API Management (`bxc-api`)

Use this skill to manage the Bxc REST/GraphQL API server (Elysia + Yoga).

## When to use
- Starting/stopping the API server.
- Managing database migrations (Drizzle).
- Checking the health and performance of the API layer.

## Commands
- `bxc api`: Start the API server.
- `bun run db:push`: Sync the database schema.
- `/bxc:api status`: Check the server status via slash command.
