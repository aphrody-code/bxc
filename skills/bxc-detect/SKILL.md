# Skill: Bxc Block & Framework Detection (`bxc-detect`)

Use this skill to identify anti-bot protections (Cloudflare, DataDome) and underlying web frameworks (Wiz, Angular, React).

## When to use
- You are being blocked by a target.
- You need to decide which profile (static, fast, stealth) to use.
- You want to identify if a site uses Google-specific technologies.

## Capabilities
- Detects 403 blocks and challenge pages.
- Identifies 5600+ Google domains and their internal frameworks.
- Suggests auto-escalation paths.
